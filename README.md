# Microservicios Productos e Inventario

Este proyecto implementa dos microservicios:

- `productos`: gestión de productos (CRUD).
- `inventario`: gestión del stock y lógica de compra.

Ambos servicios están hechos en Spring Boot y expuestos vía API REST para ser consumidos mediante Postman o entre sí.
